# LGooogle
-LG Twins 팬 블로그

## 1. 목표와 기능

+ 회원가입, 로그인, 로그아웃 기능
+ 글 쓰기는 회원만이 가능
+ 글 삭제 및 수정은 작성자만이 가능 : 작성자가 아닌 회원이 수행하려 할 시 403 Forbidden
+ 글 검색 및 조회는 비회원도 가능

## 2. 개발환경 및 사용언어, 라이브러리

+ Visual Studio Code : ver 1.81.1
+ Chromium: 108.0.5359.215
+ OS: Windows_NT x64 10.0.25336

+ python 3.12
+ Django 4.2.7
